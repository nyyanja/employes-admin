# Employes Admin

A simple HR back-office built with React-Admin to manage employees (list, create, edit, delete).

## Tech Stack
- React + TypeScript
- React-Admin
- JSON Server

## Getting Started

Terminal 1 — Start the API:
```bash
npx json-server db.json --port 3002
```

Terminal 2 — Start the app:
```bash
npm run dev
```
